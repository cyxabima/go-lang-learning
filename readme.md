### My Go Journey üöÄ

In my **3rd semester**, I took the initiative to seriously learn **Go** ‚Äî and it turned out to be a **wonderful experience**. I learned a lot, not just syntax, but *how to think differently*.

What I loved most about Go:

* An **excellent standard library**
* **Dependency management** using remote GitHub/module URLs
* **Errors as values** (no try/catch blocks)
* **Pointers**, but *no pointer arithmetic* (safe by design)
* **Concurrency** with goroutines, channels, and `WaitGroup`
  (still exploring this deeply)
* **Garbage collection**, no manual memory handling

Compared to C++, Go feels **simple to the point that it almost feels like cheating** ‚Äî but in a good way.
